## Documentation Guide

All documentation is stored in the `/docs` directory. You can find useful information about the `The-Codec` project there. The subdirectories are organized as below.

- `/guides`: Guides are collections of tutorials, references, and tips about this project. If you’re planning to start or currently developing a similar project like this, I recommend you to check out these first.
- `/inspirations`: Inspirations are short memos that help me to design architecture, looks, and lines of code for this project.
